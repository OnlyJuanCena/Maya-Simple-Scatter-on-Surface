## Maya Simple Scatter-on-Surface

In my previous project I wanted the ability to take a procederal object and scatter it across a plane, but the methods for doing so were not only complicated but maybe even over-the-top for what I was wanting.
So, I took it upon myself to make a simple tool that allows for easy scattering across not just a plane, but any basic shape object they desire. This tool is designed for beginning artists that need a simple yet effective way to utilize this basic function. It can also be used by advanced artists for quicker prototyping and revisions.

GitHub URL: https://github.com/OnlyJuanCena/Maya-Simple-Scatter-on-Surface
