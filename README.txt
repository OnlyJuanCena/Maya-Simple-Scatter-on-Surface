## Maya Simple Scatter-on-Surface

In my previous project I wanted the ability to take a procedural object and scatter it across a plane, but the methods for doing so were not only complicated but even over-the-top for what I was wanting.
So, I took it upon myself to make a simple tool that allows for artists to access easy scattering across not just a plane, but any basic shape object they desire. This tool is designed for beginning artists that need a simple yet effective way to utilize this basic function. It can also be used by advanced artists for quicker prototyping and revisions.
It's ability to be changed in real time gives an edge over other tools because it allows for even faster object viewing and iterations.

There are 3 main features with this plug-in:
Real-time changes -  allows you to make changes to the scattered objects distribution in real-time.
Automatic grouping and parenting - scattered objects are made into a child of the object they are scattered onto making for easy movement and manipulation.
Supports groups - both objects and object groups of anu kind can be scattered onto any kind of base geometry.
